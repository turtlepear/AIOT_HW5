# 📝 AI vs Human Detector（英文版）

透過這個 Streamlit 應用程式，你可以快速判斷一段 **英文文本** 是 **AI
生成** 還是 **人類撰寫**。\
介面美觀、操作簡單，並提供示範文本與視覺化結果。

demo web : https://aiothw5-fjlznakk7zjiu2nndeskks.streamlit.app/

## 🚀 功能特色

### ✔️ **英⽂文本判斷**

使用 **DistilBERT 英文情緒分類模型（SST-2）** 來推估文本的 AI/Human
特性。


### ✔️ **內建示範文本**

-   提供 *Human* 與 *AI* 類別示例
-   支持一鍵隨機生成示例文本

### ✔️ **高效模型快取**

使用 `@st.cache_resource`，首次載入後後續推論更快速。

------------------------------------------------------------------------

## 📦 安裝方式

請先建立虛擬環境（建議），並安裝必要套件：

``` bash
pip install streamlit transformers torch numpy
```

------------------------------------------------------------------------

## ▶️ 如何執行

將提供的 `.py` 檔放置專案目錄中，例如 `app.py`，然後執行：

``` bash
streamlit run app.py
```

指令執行後瀏覽器會自動開啟：

    http://localhost:8501

------------------------------------------------------------------------

## 🧠 模型說明

本程式使用：

-   **distilbert-base-uncased-finetuned-sst-2-english**
    -   預訓練於英文情緒分類（正向/負向）
    -   本專案將「POSITIVE」結果映射為較偏向 AI 風格
    -   並依照模型 score 給出 AI / Human 機率


------------------------------------------------------------------------

## 🖼️ 使用流程

1.  選擇示範文本類型（可選）
2.  可按下 **🎲 隨機示範文本**
3.  或手動輸入自己的英文文章
4.  按下 **判斷**
5.  觀察結果：
    -   AI 機率％
    -   Human 機率％
    -   條形圖比較

## 🔧 常見問題 FAQ

### 1️⃣ 網頁無法自動開啟？

手動開啟瀏覽器並輸入：

    http://localhost:8501

### 2️⃣ 模型載入很慢？

第一次下載 DistilBERT 需要一些時間，之後會自動快取。

### 3️⃣ 可以改成中文偵測嗎？

可以，但需要替換成支援中文的模型，例如： -
`uer/roberta-base-finetuned-chinanews-chinese` -
`hfl/chinese-roberta-wwm-ext`
